# Start Here

This repository contains one thing: a PowerShell installer for creating new projects from `di-sukharev/vibe`.

Start with:

1. `README.md`
2. `scripts/bootstrap-project.ps1`
3. `TASKS.md`

For every change, run a scratch install and confirm:

- project folder is created;
- root `README.md` has `Project Bootstrap Plan`;
- selected surfaces are active;
- deferred surfaces are marked as deferred;
- template `origin` is removed unless `-KeepTemplateRemote` is passed.

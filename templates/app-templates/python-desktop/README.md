# {{PROJECT_NAME}}

Simple Python desktop app starter.

## What This Is

This is a local computer application, not a website, mobile app, or browser extension.

It starts with Python standard library `tkinter` so the first version has no external UI dependency.

## Run

```powershell
python main.py
```

## Next UI Choices

Keep `tkinter` for a small utility.

Consider `customtkinter` when the app should look more modern without becoming complex.

Consider `PySide6` when the app needs a larger professional desktop UI.

Consider `Flet` when you want a Python app with a web-like UI model.

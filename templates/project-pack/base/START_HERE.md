## Daily Workflow

1. Pick one small task from `TASKS.md`.
2. Ask the AI agent to inspect first and change nothing.
3. Approve a small plan.
4. Let the agent implement.
5. Run the shortest useful check.
6. Run code review.
7. Commit only the working change.

Do not paste secrets, tokens, passwords, cookies, or `.env` contents into chat.

# Варианты Основы Проекта

Этот файл нужен, когда выбираем, что именно вайбкодить и какую основу загрузить.

## Сейчас Встроено

- `Vibe mobile + web`: приложение с web, mobile, backend/API, auth/database patterns, feature packs и deployment planning.
- `Chrome extension`: browser extension на `JohnBra/vite-web-extension`: React, TypeScript, Tailwind CSS, Vite, Manifest V3.

## Возможные Основы На Потом

- Landing page / marketing site: Astro или Next.js, красивые секции, pricing, FAQ, SEO, analytics.
- Internal tool / dashboard: shadcn/ui, tables, filters, forms, charts, auth, admin workflows.
- SaaS app: auth, billing, teams, roles, onboarding, settings, email, audit log.
- AI tool: prompt UI, upload, job status, history, usage limits, eval prompts.
- Ecommerce/catalog: products, filters, checkout planning, SEO pages, admin content flow.
- Course/membership site: lessons, progress, paywall, library, account area.
- Design system starter: tokens, buttons, inputs, cards, modals, navigation, empty/loading/error states.

## Дизайн По Умолчанию

- Web: shadcn/ui-style components, потом настройка под дизайн продукта.
- Mobile: маленькая библиотека компонентов под конкретный проект.
- Extension: сфокусированный popup/options/content-script UI и минимум permissions.

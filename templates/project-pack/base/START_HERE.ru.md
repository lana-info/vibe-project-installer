## Ежедневный Workflow

1. Выбери одну маленькую задачу из `TASKS.ru.md`.
2. Попроси AI сначала изучить контекст и ничего не менять.
3. Утверди короткий план.
4. Дай AI реализовать только эту задачу.
5. Запусти самую короткую полезную проверку.
6. Попроси code review.
7. Коммить только рабочее изменение.

Не вставляй в чат секреты, токены, пароли, cookies и содержимое `.env`.

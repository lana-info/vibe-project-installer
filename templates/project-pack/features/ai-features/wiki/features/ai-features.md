# AI Features

Use this feature pack if the product includes generation, analysis, assistants, prompt flows, summaries, recommendations, or AI automation.

## Product Questions

- What should AI do for the user?
- What user input does it need?
- What output should be saved?
- Does output need review before publishing?
- What data must never be sent to an AI provider?

## Setup Checklist

- [ ] Define AI inputs, outputs, and review flow in `PRD.md`.
- [ ] Keep API keys out of git and chat.
- [ ] Add logging without storing sensitive prompts or private data.
- [ ] Add tests around permissions and saved outputs.
- [ ] Add human review before publishing AI-generated public content when needed.
